## 📝 About

This project uses **MobileNetV2** for **image classification** to detect Nepali banknotes.  
It’s designed for **mobile applications**, making currency detection easy and fast.  

---

## 🚀 Features

- Detects **all major Nepali banknotes**.  
- Built using **TensorFlow / Keras** and **MobileNetV2**.  
- Ready for **mobile deployment**.  
- Lightweight model for **fast inference**.

## 💻 Installation

```bash
# Clone the repo
git clone https://github.com/NaranSapkota/Nepali_Currency_Detection_Using_MobileNetV2.git
cd Nepali_Currency_Detection_Using_MobileNetV2

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows

# Install dependencies
pip install -r requirements.txt
